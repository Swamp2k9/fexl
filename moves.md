# Fighgting EX Layer Move Data

## Introduction

### Version

* Dec 15 '17
* appver.0.12.0
* dataver.0.2.4

### Legend

* S : Standing
* C c. : Crouching
* f. : Front
* b. : Back

### Common

* 全体的にEXシリーズより有利フレームが1伸びているため、EXと同等フレームなら有利フレームは+1
* 立ちとしゃがみの移行の1Fがなくなったので、立ち→しゃがみの目押し猶予はさらに+1
* ヒットストップは7F
* 有利フレームの計算式: 発生 + 小(14)/中(19)/大(30)/下大P(27) - 全体
